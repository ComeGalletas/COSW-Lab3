Controlador para tareas pendientes

Esta pequeña aplicacion permite crear y controlar tareas pendientes que pueda necesitar.

Instrucciones

En el menú de arriba se encuentran las vistas

- La vista 1 corresponde a esta pagina, las instrucciones.

- La vista 2 se usa para agregar tareas pendientes. Se debe usar una prioridad de 1 a 10 para las de menor a mayor prioridad.

- La vista 3 sirve para consultar las tareas agregadas. Tambien se pueden ordenar por descripcion y por prioridad.
